# 負担割合別割り勘計算ツール

学生や新人など、負担割合が異なる参加者がいる飲み会・食事会の割り勘を簡単に計算できるNext.jsアプリです。

## 主な機能

- **負担割合の設定**: 10割（全額）から5割（半額）まで、10%刻みで設定可能
- **人数入力**: 各負担割合ごとに人数を入力
- **端数処理**: 1円、10円、100円、1,000円単位で切り上げ可能（デフォルト: 100円）
- **リアルタイム計算**: 入力と同時に支払額を自動計算
- **自動保存**: ブラウザのLocalStorageに自動保存

## 使い方

1. **合計金額を入力**: 会計の合計金額を入力
2. **端数処理を選択**: 切り上げ単位を選択（デフォルト: 100円）
3. **負担割合と人数を入力**: 各割合ごとの参加人数を入力
4. **計算結果を確認**: 各割合ごとの支払額が自動表示

## 計算ロジック

```
1. 負担ウェイトの合計 = Σ(負担割合 × 人数)
2. 基準単価 = 合計金額 ÷ 負担ウェイト合計
3. 支払額 = 基準単価 × 負担割合（端数処理適用）
```
