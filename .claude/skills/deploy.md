# デプロイ前チェック＆実行

新しいページ追加やコード変更後のデプロイ準備スキル。

## チェックリスト

1. **ビルド確認**: `npx next build` が成功するか
2. **リンク確認**: ヘッダー・フッター・トップページからのリンクが正しいか
3. **メタデータ確認**: 各ページに title / description が設定されているか
4. **レスポンシブ確認**: モバイル表示に問題がないか（Tailwindクラスの確認）
5. **ダークモード確認**: `dark:` プレフィックスが適切に使われているか

## 実行手順

```bash
# 1. ビルド
npx next build

# 2. 変更をコミット
git add .
git commit -m "変更内容の要約"

# 3. プッシュ
git push
```

## AdSense 審査チェック（新規ページ追加時）
- [ ] プライバシーポリシーが最新か (`/privacy`)
- [ ] 利用規約の「サービスの内容」に新アプリが含まれているか (`/terms`)
- [ ] 運営者情報の「提供中のアプリ」に追加されているか (`/about`)
- [ ] トップページのアプリ一覧に追加されているか (`/`)
